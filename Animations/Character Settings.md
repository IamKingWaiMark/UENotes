## Character Movement

### Orient Rotation to Movement

Enable's rotation on the character based on the movement.

If true, it will rotate the character

If false, it will prevent the character model from rotation


### Use Controller Desiredd Rotation

This option allows the character to try to rotate toward the controller’s desired facing direction.

If true, it will rotate the character to the direction the camera is facing

If falsse, it will ignore it so the camera and the character will behave indepently.
